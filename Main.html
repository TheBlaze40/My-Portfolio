Projects


Neopixel LED Strip Controlled by Computer Input

This project involves designing and implementing a system where a Neopixel LED strip is controlled via computer input. The setup includes a Neopixel LED strip connected to a microcontroller, which interfaces with a computer through a USB connection. Software on the computer sends commands to the microcontroller, allowing dynamic control over the LED strip's color, brightness, and lighting patterns. The project aims to demonstrate real-time customization of LED displays using computer-generated input, making it ideal for applications in interactive displays, decorative lighting, and visual effects.

LCD Programmed with Arduino

This project focuses on using an Arduino microcontroller to control an LCD display. The setup involves connecting a Liquid Crystal Display (LCD) to an Arduino board, enabling the display of text, numbers, and basic graphics. The Arduino is programmed to send data to the LCD, allowing for real-time updates and interactive user interfaces. Applications include creating custom displays for information, status indicators, and simple visual outputs in various electronic projects. The project highlights the integration of Arduino programming with LCD technology to deliver informative and interactive displays.
